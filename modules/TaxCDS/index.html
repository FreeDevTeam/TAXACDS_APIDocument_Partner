Partner API Module: TaxCDS

Các API tra cứu thông tin thuế dành cho đối tác.

Quay về trang chủ | Mở bản Markdown module

Danh sách API

  • Tra cứu mã số thuế MST
    POST /TaxCDSMstLookup/partner/lookupMst
    HTML | MD
  • Tra cứu tình trạng hộ kinh doanh HKD
    POST /TaxCDSMstLookup/partner/lookupHkdStatus
    HTML | MD
  • Tra cứu nghĩa vụ thuế Nghĩa vụ thuế
    POST /TaxCDSObligationLookup/partner/lookupTaxObligation
    HTML | MD
  • Tra cứu quyết toán thuế Quyết toán
    POST /TaxCDSFinalizationLookup/partner/lookupTaxFinalization
    HTML | MD
  • Tra cứu tình trạng nộp tờ khai Tờ khai
    POST /TaxCDSDeclarationStatusLookup/partner/lookupDeclarationStatus
    HTML | MD

Headers chung

  • apikey: Khóa xác thực API của đối tác.

Payload chung

Các API tra cứu TaxCDS dùng chung cấu trúc payload:

{
  "filter": {
    "taxCode": "8173748371"
  },
  "skip": 0,
  "limit": 10
}

Cần truyền ít nhất một điều kiện trong filter: taxCode, citizenIdentityNumber hoặc taxpayerName.
